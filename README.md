# 로셜리티 입찰검색 대시보드

나라장터(조달청) 입찰공고 API를 활용해 키워드별 맞춤 공고를 조회하는 정적 대시보드입니다.

## 제작
- 제작: 임팩트브랜딩커뮤니티 (주)로셜리티

## 실행 방법
브라우저에서 `index.html`을 바로 열거나, 간단한 정적 서버로 실행하세요.

```bash
python3 -m http.server 8080
```

이후 `http://localhost:8080` 접속.

## 주요 기능
- 기본 키워드 제공: `로컬, 문화, 연구, 청년, 창업, 지역, 상권, 기본계획, 운영, 활성화`
- 키워드 추가/삭제 가능(로컬 저장)
- 키워드별 최신 검색결과 10건 표시
- 참가가능지역이 `서울`, `경기`, `전국`인 항목만 노출
- 컬럼: 공고명 / 금액 / 공고기관 / 입찰마감일시 / 파일첨부문서
- 60초 간격 자동 새로고침 + 수동 새로고침

## API 정보
- 서비스: 나라장터 `getBidPblancListInfoServcPPSSrch`
- 인증키: 코드에 반영됨(요청값)

> 참고: 공공데이터 API CORS/망 정책에 따라 브라우저 단독 호출이 제한될 수 있습니다.
> 이 경우 서버 프록시(백엔드) 연결이 필요합니다.
